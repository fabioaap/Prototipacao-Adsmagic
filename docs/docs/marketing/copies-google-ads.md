# Kit de Copies — Google Ads (Search)

Kit operacional de copies para a campanha **Captura de intenção — LP Agencias** no Google Ads (Search). Contém mapa de palavras-chave, anuncios responsivos (RSA), extensões e recomendações de campanha.

---

## Propósito e regras de uso

**Produto:** Adsmagic — plataforma de atribuição, inteligência e envio de conversões para agências que vendem via WhatsApp.

**ICP:** Donos de agência de performance/growth/tráfego pago e gestores de tráfego que operam contas com vendas via WhatsApp.

**Pricing vigente:** R$ 147/mês.

**CTA principal:** Demo diagnóstica.

**LP de destino:** `/lp/agencias`

**Tipo de campanha:** Search (captura de intenção — quem pesquisa já tem consciência do problema).

**Mensagem-mãe:** Pare de reportar lead. Comece a provar venda.

**Regra editorial para claims:**

- Claims sustentados podem ser usados como afirmação direta.
- Claims parcialmente sustentados devem usar linguagem como "desenhado para", "permite que", "torna possível". Nunca promessas absolutas.
- Sem emojis em nenhuma copy.

> **Anti-categoria:** O Adsmagic nao é CRM, nao é chatbot, nao é automação ampla e nao é tracker genérico. Evitar qualquer headline ou description que posicione o produto nessas categorias.

## Recomendações de campanha

### Tipo de match por grupo

| Grupo | Match principal | Observação |
|-------|-----------------|------------|
| A — Atribuição | Phrase | Intenção clara; phrase preserva relevância |
| B — CPL/Lead | Broad | Intenção difusa; broad com Smart Bidding captura variações |
| C — Concorrência | Exact + Phrase | Proteger marca do concorrente; exact para termos diretos |
| D — Categoria | Phrase + Broad | Misto — termos técnicos em phrase, variações em broad |

### Bidding strategy

**Recomendação:** Maximize Conversions com Target CPA.

- Fase de aprendizado (primeiras 2-4 semanas): usar Maximize Clicks com orçamento controlado para acumular dados de conversão.
- Após 30-50 conversões acumuladas: migrar para Maximize Conversions com Target CPA.
- Conversão principal a configurar: envio do formulário de demo diagnóstica na LP `/lp/agencias`.

### Budget diário sugerido

| Fase | Budget diário | Observação |
|------|---------------|------------|
| Aprendizado (semanas 1-4) | R$ 50–80 | Acumular dados de conversão e quality score |
| Escala inicial (semanas 5-8) | R$ 100–150 | Ajustar com base em CPA real observado |
| Escala madura (semana 9+) | Definir com base em CPA target | Escalar grupos com melhor desempenho |

> Budget total estimado para fase de aprendizado: R$ 1.500–2.400/mês.

### Landing page

- **URL de destino:** `/lp/agencias`
- **Parâmetros UTM sugeridos:** `utm_source=google&utm_medium=cpc&utm_campaign={campaign_name}&utm_content={ad_group}&utm_term={keyword}`
- **Tracking:** configurar Google Ads conversion tag + evento de formulário como conversão primária.
- **Coerência:** garantir que a LP repita os termos principais das headlines para manter quality score alto.

### Estrutura de campanha sugerida

```
Campanha: Adsmagic — Search — Agências
├── Grupo A: Atribuição
│   ├── RSA (Atribuição)
│   └── Keywords de atribuição (phrase)
├── Grupo B: CPL/Lead
│   ├── RSA (CPL/Lead)
│   └── Keywords de problema (broad)
├── Grupo C: Concorrência
│   ├── RSA (Concorrência)
│   └── Keywords de concorrência (exact + phrase)
└── Grupo D: Categoria
    ├── RSA (Categoria)
    └── Keywords de categoria (phrase + broad)
```

### Considerações adicionais

- **Negativar entre grupos:** adicionar keywords de cada grupo como negativas nos outros para evitar canibalização.
- **Audiência de observação:** adicionar audiência personalizada de visitantes da LP para observação (não targeting) e ajuste de lance.
- **Dispositivo:** sem restrição de dispositivo na fase de aprendizado; ajustar bids por device após dados suficientes.
- **Geolocalização:** Brasil inteiro na fase inicial; refinar se houver concentração de conversões por região.
- **Horário:** sem restrição inicial; aplicar ad schedule após 4+ semanas de dados.
