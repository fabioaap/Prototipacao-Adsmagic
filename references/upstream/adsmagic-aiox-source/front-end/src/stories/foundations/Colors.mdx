# Colors

O sistema de cores do Adsmagic Design System é baseado em tokens semânticos que se adaptam ao tema (claro/escuro).

## Cores Primárias

As cores primárias são usadas para elementos de destaque e ações principais.

## Uso em Código

### CSS Variables

```css
.meu-componente {
  background-color: hsl(var(--primary));
  color: hsl(var(--primary-foreground));
  border: 1px solid hsl(var(--border));
}
```

### TailwindCSS

```html
<div class="bg-primary text-primary-foreground border-border">
  Conteúdo
</div>
```

### Cores Semânticas no Tailwind

```html
<!-- Sucesso -->
<span class="text-success bg-success/10">Ativo</span>

<!-- Aviso -->
<span class="text-warning bg-warning/10">Pendente</span>

<!-- Erro -->
<span class="text-destructive bg-destructive/10">Erro</span>

<!-- Info -->
<span class="text-info bg-info/10">Novo</span>
```

## Tokens de Cor

| Token | Uso | Exemplo |
|-------|-----|---------|
| `--background` | Fundo da página | `bg-background` |
| `--foreground` | Texto principal | `text-foreground` |
| `--card` | Fundo de cards | `bg-card` |
| `--card-foreground` | Texto em cards | `text-card-foreground` |
| `--primary` | Ações principais | `bg-primary` |
| `--primary-foreground` | Texto em botões primários | `text-primary-foreground` |
| `--secondary` | Ações secundárias | `bg-secondary` |
| `--muted` | Fundos sutis | `bg-muted` |
| `--muted-foreground` | Texto secundário | `text-muted-foreground` |
| `--accent` | Hovers e destaques | `bg-accent` |
| `--destructive` | Erros e ações perigosas | `bg-destructive` |
| `--success` | Status positivos | `bg-success` |
| `--warning` | Alertas | `bg-warning` |
| `--info` | Informações | `bg-info` |
| `--border` | Bordas | `border-border` |
| `--input` | Bordas de inputs | `border-input` |
| `--ring` | Focus rings | `ring-ring` |
