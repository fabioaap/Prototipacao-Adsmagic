## Uso em Código

### Classes TailwindCSS

```html
<!-- Headings -->
<h1 class="text-4xl font-bold">Título Principal</h1>
<h2 class="text-2xl font-semibold">Subtítulo</h2>
<h3 class="text-xl font-semibold">Card Title</h3>

<!-- Body -->
<p class="text-base">Texto padrão</p>
<p class="text-sm text-muted-foreground">Texto secundário</p>
<span class="text-xs text-muted-foreground">Caption</span>

<!-- KPI Values -->
<span class="text-3xl font-bold tracking-tight">R$ 45.000</span>
<span class="text-sm text-muted-foreground">vs. mês anterior</span>
```

### Componentes Comuns

| Elemento | Classes | Exemplo |
|----------|---------|---------|
| Page Title | `text-3xl font-bold tracking-tight` | Dashboard |
| Section Title | `text-xl font-semibold` | Métricas |
| Card Title | `text-lg font-medium` | Contatos Novos |
| KPI Value | `text-3xl font-bold` | 1.234 |
| KPI Label | `text-sm text-muted-foreground` | Este mês |
| Body Text | `text-base` | Descrição... |
| Helper Text | `text-sm text-muted-foreground` | Ajuda |
| Badge | `text-xs font-medium` | Ativo |
| Button | `text-sm font-medium` | Salvar |
