# landing-pages

App dedicado para as landing pages exportaveis do AdsMagic Workspace.

## Scripts

- npm run dev
- npm run build
- npm run preview
- npm run package
- npm run build:deliverables

## Estrutura

- src/catalog: pagina raiz de preview interno
- src/lps/vendas-whatsapp: LP Adsmagic para Vendas no WhatsApp
- src/lps/agencias-performance: LP Adsmagic para Agencias de Performance
- src/shared: reservado para extracoes futuras
- public/img/landing: assets publicos das LPs
- scripts/package-lps.mjs: gera deliverables estaticos por LP
